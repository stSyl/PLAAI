## Task:
The AI Co-coding pattern employs the AI's understanding of syntax and process flow to aid in the coding process. This pattern can effectively speed up development, help identify potential errors, and suggest alternative solutions. However, the AI's suggestions are only as good as the context it is provided, and it may not account for every possible edge case or high-level design considerations. For this pattern to work well, the user must provide clear, precise instructions, and also critically review and test the AI's contributions.
